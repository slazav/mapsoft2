Mapsoft2

Набор библиотек и программ для работы с геоданными и картами.

О проекте

Мы начали писать программы для работы с геоданными и картами в 1997-1998 годах, когда в нашей походной компании появились первые GPS-приемники. Сперва это были программы для скачивания треков, рисования их на карте, редактирования и загрузки в GPS. Потом мы начали рисовать свои собственные карты: походные карты Подмосковья и некоторых горных районов. Все программы для работы с геоданными, растровыми и векторными картами, SRTM-данными и т.п. были собраны в проект mapsoft. Mapsoft2 -- это попытка аккуратно переписать эти программы, исправляя разные ошибки, заложенные в архитектуре, и улучшая код.

man-страницы (EN)

  •   ms2conv -- Преобразования геоданных. Форматы: GPX, KMZ/KML, OziExplorer, GeoJSON, GarminUtils, ZIP.
  •   ms2geofig -- Работа с FIG-файлами, имеющими геодезическую привязку. Добавление/извлечение растровых карт, добавление геоданных, DEM-данных.
  •   ms2img -- Преобразования растровых изображений.
  •   ms2mp -- Преобразования MP файлов. Сделано в основном для тестирования mapsoft2.
  •   ms2nom -- Программа для вычислений, связанных с советской номенклатурой карт: координаты данного листа, соседние листы, покрывающие данный район листы и т.п.
  •   ms2proj -- Преобразование геодезических координат.
  •   ms2render -- Рендер векторных и растровых карт, геоданных, DEM-данных.
  •   ms2tiles -- Программа для вычислений, связанных с тайлами TMS/Google: координаты данного тайла, покрывающие данный район тайлы и т.п.
  •   ms2view -- Вьюер для геоданных и растровых карт.
  •   ms2vmap -- Преобразование векторных карт. Форматы: vmap2db, vmap2, vmap, mp, fig.
  •   ms2vmapdb -- Работа с базой данных векторных карт (vmap2db).
  •   ms2xyz -- Программа для вывода треков в виде текстовой таблицы. Вычисления времен, расстояний, скоростей, настраиваемый формат.

Здесь   - более-менее стабильные программы,   - нестабильные программы, интерфейс которых будет меняться,   - недоделанные программы, которые пока нельзя использовать.
